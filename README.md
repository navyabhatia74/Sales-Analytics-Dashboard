# Sales-Analytics-Dashboard
## Overview
This Sales Dahsboard was created using Microsoft Excel to analyze sales performance , profit trends, category-wise revenue, and regional business insights through interactive visualizations
## Key Insights
-Revenue trend analysis 
-Profit performance tracking 
-Category-wise sales comparison 
-Regional sales distribution
-Order and quantity metrics
## Dashboard Features
-KPI cards
-Revenue trend visualization
-Regional analytics
-Interactive slicers
-Donut and bar charts
## Tools used 
-Microsoft Excel
-Pivot Tabels
-Pivot Charts
-Data Visualization
